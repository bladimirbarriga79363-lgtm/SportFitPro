SportFit Pro
Bienvenido al repositorio oficial de SportFit Pro, un prototipo de sitio web estático y responsivo enfocado en la venta y administración de artículos deportivos.

Este proyecto fue desarrollado como parte de la Unidad 2: HTML, XML y CSS de la materia de Programación Web.

Características del Proyecto:
Estructura Semántica: Construido con HTML5 siguiendo las mejores prácticas y etiquetas descriptivas.

Diseño Responsivo y Modular: Uso de CSS3 (Modelo de Caja y Flexbox) con hojas de estilo separadas para el diseño base y los formularios.

Panel de Administración: Interfaz para la gestión de inventario (Altas, Modificaciones y Bajas) simulando el método GET.

Calidad Certificada: Todo el código fuente cuenta con la validación oficial del W3C (World Wide Web Consortium) sin errores ni advertencias.

Tecnologías Utilizadas:
HTML5

CSS3

Git y GitHub (Control de versiones en equipo)
